Pasos de instalación

  1. Descarga el instalador.
  2. Ejecuta el archivo.
  3. Sigue el manual de usuario.
  4. Si Windows o Chrome muestra una advertencia, revisa las instrucciones de seguridad.

Si Chrome bloquea la descarga

Es posible que Chrome muestre una advertencia porque ClipMind AI es un software nuevo y todavía no tiene muchas descargas registradas. Esto puede pasar con archivos .exe nuevos.

Antes de continuar, verifica esto:
  • El archivo debe llamarse Instalador_ClipMindAI.exe.
  • Debes descargarlo únicamente desde esta página oficial.
  • No descargues el instalador desde enlaces desconocidos o reenviados por terceros.

Pasos para conservar la descarga en Chrome

  1. Haz clic en el ícono de Descargas, arriba a la derecha de Chrome. Normalmente se ve como una flecha hacia abajo.
  2. Busca el aviso relacionado con Instalador_ClipMindAI.exe.
  3. Haz clic en Más información o en el menú de opciones de esa descarga.
  4. Elige Conservar de todos modos, Mantener archivo o Continuar descarga. El texto puede variar según tu versión de Chrome.
  5. Espera a que la descarga termine completamente.
  6. Cuando termine, abre el archivo descargado para iniciar la instalación.

Si no ves esas opciones, intenta hacer clic en los tres puntos del aviso de descarga o abre la carpeta de descargas de tu computadora.

Si Windows muestra una alerta azul

Windows puede mostrar una ventana llamada SmartScreen con un mensaje como “Windows protegió tu PC”. Esto puede aparecer porque ClipMind AI es un programa nuevo y el editor aún no tiene reputación suficiente en Windows.

Antes de ejecutarlo:
  • Confirma que el archivo se llama Instalador_ClipMindAI.exe.
  • Confirma que lo descargaste desde esta página oficial.
  • Si tienes dudas, contacta a soporte antes de abrirlo.

Pasos para ejecutarlo en Windows

  1. Haz doble clic sobre Instalador_ClipMindAI.exe.
  2. Si aparece una ventana azul de Windows, haz clic en Más información.
  3. Luego aparecerá el botón Ejecutar de todas formas.
  4. Haz clic en Ejecutar de todas formas solo si reconoces que descargaste el instalador desde la web oficial de ClipMind AI.
  5. Sigue los pasos del instalador hasta finalizar.

Estas advertencias son preventivas. ClipMind AI es un software nuevo, por eso algunos navegadores o Windows pueden pedir confirmación adicional antes de abrirlo.
